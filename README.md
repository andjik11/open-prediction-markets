# Open Prediction Markets

Open Prediction Markets is an open-source Web3 public-good initiative focused on building transparent prediction market infrastructure for **forecasting, research, and education**.

## Why prediction markets?
Prediction markets aggregate collective intelligence and can improve forecasting quality when designed transparently and responsibly.

This project aims to explore:
- Open and auditable market mechanisms
- Responsible market design (research & education focus)
- Oracle and dispute resolution approaches
- Community governance experiments

## Current status
- Research & market design phase
- Requirements and architecture planning
- Preparing MVP scope for testnet experimentation

## Roadmap
1. **Phase 1:** Research & market design + documentation
2. **Phase 2:** MVP smart contracts (testnet) + basic UI prototype
3. **Phase 3:** Oracle & dispute mechanisms + security review
4. **Phase 4:** Public beta + governance experiments
